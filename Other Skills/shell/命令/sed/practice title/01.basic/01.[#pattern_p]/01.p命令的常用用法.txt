title：
  p命令用于打印sed处理结果
  
summary:
  [1] 规定在默认情况下，sed命令会将模式空间中的内容打印到标准输出一次；在此情况下不需要指定p命令。
  example no.1:
  - #BASH> echo -e 'line 1\nline 2\nline 3'|sed ''
  
  - line 1
  - line 2
  - line 3
  
  example no.2:
  - #BASH> echo -e 'line 1\nline 2\nline 3' | sed 's/1/n/'
  
  - line n
  - line 2
  - line 3
  
  [2] 被模式匹配到的内容如果被命令动作处理过，一般情况下会再次打印到标准输出。此情况需要指定p命令。
  example no.1:
  - #BASH> echo -e 'line 1\nline 2\nline 3' | sed 's/1/n/p'
  
  - line n <<此行内容应该是再次打印的被命令处理过后的匹配内容
  - line n <<此行内容是模式空间中的内容
  - line 2 <<此行内容是模式空间中的内容
  - line 3 <<此行内容是模式空间中的内容
  
  [3] -n选项代表不输出模式空间中的内容至屏幕，即关闭不能被模式匹配到的行到标准输出中。
      此选项与p命令配合，可实现只打印被命令处理过的匹配内容的效果。
 example no.1:
 - #BASH> echo -e 'line 1\nline 2\nline 3' | sed -n 's/1/n/p'
 
 - line n <<只输出p命令的输出
